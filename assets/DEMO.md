# Demo GIF Workflow

1. Record a short screen demo (10-20 seconds) and save it as:
   - `assets/demo.mov`
2. Install ffmpeg:
   - `brew install ffmpeg`
3. Generate GIF:
   - `./scripts/make-demo-gif.sh`
4. Commit `assets/demo.gif` so it appears in README.

Tip: Keep the recording focused on one flow (generate -> copy share -> open history -> export PDF).
